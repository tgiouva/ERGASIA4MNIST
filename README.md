Στόχος αυτής της εργασίας είναι η δημιουργία και εκπαίδευση ενός νευρωνικού δικτύου για την αναγνώριση χειρόγραφων ψηφίων.Το MNIST περιέχει εικόνες ψηφίων από το 0 έως το 9 μεγέθους 28x28 pixels.Το ζητούμενο ήταν το μοντέλο να μπορεί να προβλέπει σωστά σε ποια κλάση ανήκει κάθε εικόνα.Για την υλοποίηση χρησιμοποίησα Python και τη βιβλιοθήκη TensorFlow.Αρχικά φορτώθηκαν τα δεδομένα του MNIST τα οποία αποτελούνται από 60.000 εικόνες εκπαίδευσης και 10.000 εικόνες ελέγχου και έγινε επεξεργασία των δεδομένων.Οι εικόνες μετατράπηκαν από μορφή 28x28 σε διανύσματα 784 τιμών για να μπορούν να δοθούν ως είσοδος στο νευρωνικό δίκτυο.Αρχικά εκπαιδεύτηκε ένα βασικό μοντέλο με δύο κρυφά επίπεδα των 256 νευρώνων,συνάρτηση ενεργοποίησης tanh,optimizer SGD,learning rate 0.001 και 10 epochs. Το μοντέλο αυτό πέτυχε validation accuracy περίπου 93.41%.Στη συνέχεια δοκιμάστηκαν διαφορετικοί συνδυασμοί παραμέτρων,όπως διαφορετική συνάρτηση ενεργοποίησης,αριθμός νευρώνων,αριθμός επιπέδων,learning rate και epochs.Τα αποτελέσματα τα αποθήκευσα σε πίνακα ώστε να συγκριθούν πιο εύκολα οι διαφορετικές δοκιμές.Από όλες τις δοκιμές,καλύτερη απόδοση είχε το μοντέλο με δύο κρυφά επίπεδα των 512 νευρώνων,ενεργοποίηση relu,learning rate 0.001 και 20 epochs με validation accuracy περίπου 96.22%.Επιπλέον,έγινε έλεγχος των λανθασμένων προβλέψεων.Το μοντέλο έκανε προβλέψεις στο test set και εμφάνισε ένα λάθος παράδειγμα για κάθε ψηφίο από το 0 έως το 9.Τέλος,υλοποίησα μία βελτίωση με χρήση Dropout(0.3).Το τελικό μοντέλο είχε δύο κρυφά επίπεδα των 512 νευρώνων με relu, Dropout μετά από κάθε κρυφό επίπεδο και έξοδο με 10 νευρώνες και softmax.Το Dropout χρησιμοποιήθηκε για να μειωθεί η υπερπροσαρμογή.Το τελικό μοντέλο πέτυχε περίπου 96.73% train accuracy και 97.12% validation accuracy.

Τελικό Μοντέλο:
Input(784),Dense(512,relu),Dropout(0.3),Dense(512,relu),Dropout(0.3),Output(10,softmax)
Optimizer:SGD
Learning rate:0.001
Epochs:20
Train Accuracy: 96.73%
Validation Accuracy: 97.12%

Συμπερασματικά,φάνηκε ότι η επιλογή των υπερπαραμέτρων επηρεάζει αρκετά την απόδοση του νευρωνικού δικτύου και με τις κατάλληλες αλλαγες,τη χρήση dropout,το μοντέλο κατάφερε να πετύχει καλύτερη ακρίβεια στην ταξινόμηση των ψηφίων.
